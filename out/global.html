converterParaISO(dataStr) → {string}

Converte uma string de data em vários formatos para o formato ISO 'YYYY-MM-DD'. A função tenta analisar uma série de formatos de data comuns. Se a data não puder ser analisada, 'Data inválida' é retornada.
Parameters:
Name Type Description
dataStr string A string de data a ser convertida.

Returns:
A data no formato 'YYYY-MM-DD' se a string de entrada for válida, ou 'Data inválida' caso contrário.
